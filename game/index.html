World layer demo

Godot 4 voxel world

A single-chunk medieval slice built in Godot 4 (GDScript): hand-authored terrain, fog and day/night, first- or third-person exploration, torches and ruins, and a sealed dungeon door with a short hint when you approach. Export targets: Web (below) and Windows from the project in game/world-demo/.

  • Controls: WASD, Space jump, mouse look (click canvas to capture pointer), Esc to release cursor, V toggles third person.
  • Embed: the iframe loads /game/play/index.html — run a Web export from Godot into game/play/ to replace the placeholder with the WASM build.
  • Desktop: use the Windows export preset (outputs next to game/world-demo/).

If you see export instructions inside the frame, the Web build has not been exported yet. Open game/world-demo in Godot, install templates, then Project → Export → Web to this folder.
